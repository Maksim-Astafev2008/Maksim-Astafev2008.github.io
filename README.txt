Maksim Astafyev — Portfolio (Neon)
==================================

• Static site (no backend). Works on GitHub Pages.
• Full RU/KZ/EN i18n (switch without page reload).
• Neon cursor, light/dark theme, animated counters.
• Testimonials with 5★ and smooth reveal.
• Certificates: open/download buttons.
• GitHub repos are loaded via GitHub API.

Structure:
  index.html
  style.css
  app.js
  certificate-845795.pdf            (optional / include next to index.html)
  Астафьев Максим Петрович.pdf      (optional / include next to index.html)

How to run locally:
  1) Just double-click index.html (or via any static server).

GitHub Pages:
  1) Create repo -> Upload all files from this folder (including PDFs).
  2) Settings -> Pages -> Source: "Deploy from a branch", Branch: "main" (root).

Notes:
  • If a PDF is missing, the viewer frame will be empty for that item.
